# Real Estate Website

This is a modern real estate website built with [Next.js](https://nextjs.org), designed to showcase property portfolios, company services, testimonials, and more. The project features a beautiful UI, responsive design, and interactive components for an engaging user experience.

## Features

- Home page with hero section and company introduction
- Services section for commercial, industrial, and residential offerings
- Architect profile and team showcase
- Portfolio gallery of completed projects
- News and articles section
- Testimonials carousel with customer feedback
- Brand partners display
- Footer with company info, navigation, and social links
- Fully responsive and mobile-friendly

## Tech Stack

- Next.js 15
- React 19
- Tailwind CSS v4
- Embla Carousel (for testimonials)
- TypeScript

## Getting Started

First, clone the repository:

```bash
git clone https://github.com/gauravshuklarwit/real-estate-website
```

Then, install dependencies and run the development server:

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `src/app/page.tsx`. The page auto-updates as you edit the file.

## Folder Structure

- `src/components/` — All UI and layout components
- `src/app/` — Main app entry and global styles
- `public/` — Images and static assets

## Deployment

Deploy easily on [Vercel](https://vercel.com/) or your preferred platform. See [Next.js deployment documentation](https://nextjs.org/docs/app/building-your-application/deploying) for more details.
